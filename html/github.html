Lista de Comandos

Comando Descripción
git init Inicializa un nuevo repositorio de Git.
git clone Copia un repositorio remoto a tu máquina local.
git add . Agrega todos los cambios al área de preparación.
git commit -m "mensaje" Realiza un commit de los cambios en el área de preparación.
git push Envía tus cambios al repositorio remoto.
